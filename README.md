# ⚽ KickOff 2026 - FIFA World Cup Prediction League

A modern and fully interactive **FIFA World Cup 2026 prediction platform** built with **React, TypeScript, TanStack Router, Tailwind CSS, and Framer Motion**.

Users can predict match scores, compete on the leaderboard, track live fixtures, and experience a premium football-themed interface with smooth animations, responsive layouts, and dynamic prediction tracking.

---

## 🚀 Features

- ⚽ Predict scores for upcoming FIFA World Cup 2026 matches
- 🏆 Real-time leaderboard with points and accuracy tracking
- 🔥 Featured match hero section with countdown timer
- 📊 Match prediction system with lock after kickoff
- 🎯 User prediction history and performance tracking
- 📺 Live match and finished match status support
- 🌑 Premium dark football-inspired UI
- ✨ Smooth animations using Framer Motion
- 📱 Fully responsive on desktop, tablet, and mobile
- 💾 Persistent local storage data handling
- 🧠 Dynamic routing with TanStack Router
- 🎨 Glassmorphism cards and neon-accent styling

## ⚙️ Installation & Setup

Follow these steps to run the project locally on your machine.

### 1. Clone the Repository

```bash
git clone https://github.com/Joras-Latihd/Mission_WorldCup_2026.git
```

---

### 2. Open the Project Folder

```bash
cd Mission_WorldCup_2026
```

---

### 3. Install Dependencies

```bash
npm install
```

---

### 4. Start Development Server

```bash
npm run dev
```

---

### 5. Open in Browser

After running the dev server, open:

```txt
http://localhost:3000
```

or the URL shown in your terminal.

# 🧠 Prediction Scoring Logic

Prediction points are calculated using:

| Prediction Result | Points |
|---|---|
| Exact score | High points |
| Correct winner | Medium points |
| Wrong prediction | 0 points |
